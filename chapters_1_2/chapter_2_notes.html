Chapter 2: Graphs and Functions

2.1 The Rectangular Coordinate System and Graphs

First Principles: The Cartesian plane maps algebraic equations to geometric shapes. The Distance Formula is just the Pythagorean Theorem in disguise: \(d = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}\).

Midpoint Formula: The average of the x and y coordinates: \((\frac{x_1 + x_2}{2}, \frac{y_1 + y_2}{2})\).

2.2 Circles

First Principles: A circle is the set of all points exactly \(r\) distance from a center \((h, k)\). Squaring the distance formula gives the standard equation.

Standard Form: \((x - h)^2 + (y - k)^2 = r^2\)

To convert from general form to standard form, you must Complete the Square for both x and y.

2.3 Lines

First Principles: The slope of a line represents its rate of change. It's the ratio of the vertical change to the horizontal change. Parallel lines have equal slopes. Perpendicular lines have negative reciprocal slopes.

Point-slope form: \( y - y_1 = m(x - x_1) \)

2.4 Relations and Functions

First Principles: A function is a rule that assigns exactly ONE output to every input. Graphically, it passes the Vertical Line Test.

Domain: The set of all valid inputs (watch out for dividing by zero or taking even roots of negative numbers).

Range: The set of all resulting outputs.

2.5 Properties of Functions

First Principles: Functions can be increasing, decreasing, or constant on an interval. They can also have symmetry.

Even Functions: \(f(-x) = f(x)\). Symmetric about the y-axis.

Odd Functions: \(f(-x) = -f(x)\). Symmetric about the origin.

2.6 A Library of Functions

First Principles: Memorize the parent graphs! The identity \(y=x\), the squaring function \(y=x^2\), the cubing function \(y=x^3\), the absolute value \(y=|x|\), the square root \(y=\sqrt{x}\), and the reciprocal \(y=1/x\).

2.7 Transformations of Functions

First Principles: Modifying the input \(x\) translates/stretches the graph horizontally. Modifying the output \(f(x)\) translates/stretches the graph vertically.

Interactive Visualization

Observe how \(a, h,\) and \(k\) transform the parent function \(f(x) = |x|\) in \(y = a|x - h| + k\).

2.8 Combining Functions; Composite Functions

First Principles: Functions can be combined algebraically (\(f+g\), \(f-g\), \(fg\), \(f/g\)) or composed (\(f \circ g\)). Composition means plugging the output of \(g\) directly into \(f\).
Gotcha Example: \((f \circ g)(x) \neq (g \circ f)(x)\). Function composition is NOT commutative!

2.9 Inverse Functions

First Principles: An inverse function completely undoes the original function. Graphically, it is a reflection across the line \(y = x\).
Gotcha Example: Does \(f(x) = x^2\) have an inverse?
Many say yes, \(y = \pm \sqrt{x}\). But \(\pm\) makes it NOT a function! A function must be One-to-One (pass the Horizontal Line Test) to have an inverse. Thus, \(x^2\) only has an inverse if we restrict its domain to \(x \ge 0\).
